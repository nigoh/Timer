# Product Overview

Focuso は、個人作業と会議進行の時間管理を一元化する業務効率化タイマーアプリ。
時間超過の抑制と作業集中の維持を支援し、会議の可視化と振り返りを促進する。

## Core Capabilities

- **集中タイマー**: カウントダウン・ポモドーロ・複数タイマーで個人の時間管理を支援
- **会議進行管理**: アジェンダ単位の時間追跡、超過検知、音声文字起こし
- **会議レポート**: 会議結果の構造化記録、AI アシスト提案、GitHub Issue 連携
- **分析ダッシュボード**: 集中時間・ポモドーロ達成率・会議超過率の可視化
- **タスクベースキャンバス**: タスクごとにウィジェットを自由配置し、作業コンテキストを保持

## Target Use Cases

- 個人の作業時間の計測・集中維持（ポモドーロ / 基本タイマー）
- 会議のファシリテーション（アジェンダ進行・時間配分・議事録記録）
- 会議後の振り返り・レポート作成（AI ドラフト生成・GitHub Issue 投稿）
- 日次〜月次での生産性パターン分析（ヒートマップ・トレンド比較）

## Value Proposition

- ブラウザ完結（インストール不要、外部 API キーはオプション）
- タスクの並び替え＋ウィジェットキャンバスで、個人にカスタマイズ可能な作業画面
- 会議データから自動でレポート草稿を生成し、振り返りコストを最小化
- タイマー・会議・分析が同一アプリ内で完結し、コンテキストスイッチを削減
