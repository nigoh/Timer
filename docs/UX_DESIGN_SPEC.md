# UXデザイン仕様（参照版）

> このドキュメントは参照用です。UI/UX要件の正本は `docs/REQUIREMENTS.md` を参照してください。

## 1. デザインシステム方針
- 現行UI方針は **shadcn/ui + Tailwind CSS** に統一。
- 既存・新規コンポーネントは上記方針に従う。
- MUI記述は過去検討履歴としてのみ扱い、現行設計の規範にしない。

## 2. 主要画面
- 基本タイマー
- ポモドーロ
- アジェンダ
- 複数タイマー

※機能定義・受け入れ条件は `docs/REQUIREMENTS.md` を優先。

## 3. 操作活性ルール（タイマー共通）
- 停止ボタンは待機中に無効化し、`Button` コンポーネント標準の `disabled` スタイル/フォーカス挙動に従う。
- 基本タイマー・ポモドーロは「実行中/一時停止中」で停止可能、アジェンダは「実行中」のみ停止可能。
