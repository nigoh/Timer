## 3. 操作活性ルール（タイマー共通）
- 「一時停止」は経過時間を保持する操作で、次の主操作ボタンは「再開」ラベルに切り替える。
- 「セッション完了」は現在の議題を完了として確定し、次の議題へ進める。
- アジェンダは「実行中/一時停止中/超過中」でセッション完了を許可する。
- アジェンダの遷移導線は「次へ」のみ提供し、実行中は無効化して停止中のみ遷移を許可する。
- ヘッダーから「＋新しい会議」「会議名を編集」を押下すると会議ダイアログが開き、会議を追加/編集できる。
- アジェンダの「次へ」は未開始（`pending`）議題では無効化し、開始済み（`running` / `paused` / `overtime`）の議題のみ完了遷移できる。
- 各議題は議事録入力欄を持ち、`Rich Text` / `Markdown` を切り替えて編集できる。
- アジェンダ一覧は右側パネルに表示し、トグルで開閉できる。モバイルではドロワー表示に切り替える。
- 各議題に「未開始 / 進行中 / 終了 / 保留」のセクション状態を表示する。
