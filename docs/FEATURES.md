# FEATURES

## 画面構成
- タブ: 基本タイマー / ポモドーロ / アジェンダ / 複数タイマー
- 共通: ログビューア、エラーバウンダリ、フッター

## 機能一覧
### 基本タイマー
- 時間設定、開始/一時停止/停止/リセット
- セッション履歴管理（削除/全削除）

### ポモドーロ
- 作業/短休憩/長休憩の状態遷移
- サイクル管理、統計表示
- 自動開始設定

### アジェンダタイマー
- 会議・議題の CRUD
- 議題の並べ替え、進行、超過判定
- 会議設定（通知・ベル・サイレント）
- 会議前ファシリテーション準備（目的種別、期待アウトプット、合意条件、事前コメント）
- 会議前ブリーフ生成（決定事項候補・争点・未解決事項）
- Parking Lot 管理（未消化論点の次回繰越）

### 複数タイマー
- 複数タイマーの同時管理
- カテゴリ管理
- 個別操作・一括操作

### 会議レポート
- 会議データからレポート草稿を生成
- ToDo 管理
- Markdown 出力

### ログ管理
- レベル別ログ記録（ERROR/WARN/INFO/DEBUG/TRACE）
- LocalStorage 保持
- AI 解析用プロンプト生成
