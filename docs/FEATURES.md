# FEATURES

## 画面構成

- 左サイドバー（shadcn Sidebar、`collapsible=icon` モード、開閉トグル付き）: タスク一覧（ドラッグ並べ替え対応）、タスク追加、設定・ログページへのリンク、ダークモード切り替え
- デスクトップ: パンくずナビ（Breadcrumb）でタスク名/設定画面を表示
- モバイル: ヘッダーバーで画面名を表示
- タスクごとにウィジェットキャンバス（初期ロード時 Skeleton 表示）: 基本タイマー / ポモドーロ / 複数タイマー / 分析 / アジェンダ / 議事録 / レポート等を自由に配置
- コマンドパレット (Ctrl+K / Cmd+K): タスク検索・切り替え、新規タスク作成、設定表示、カラーモード切り替え
- トースト通知 (Sonner): 保存完了・コピー完了等のフィードバック
- 共通: ログビューア、エラーバウンダリ、フッター

## 機能一覧

### 基本タイマー

- 時間設定、開始/一時停止/停止/リセット
- セッション履歴管理（削除/全削除）
- 履歴エントリごとの GitHub Issue 紐付け（追加/解除）
- Issue タイトル自動取得（Owner/Repo + Issue番号入力時、GitHub API 利用）
- GitHub Issue 別の合計作業時間/件数の集計表示

### ポモドーロ

- 作業/短休憩/長休憩の状態遷移
- サイクル管理、統計表示
- 自動開始設定

### アジェンダタイマー

- 会議・議題の CRUD
- 議題の並べ替え、進行、超過判定
- 会議設定（通知・ベル・サイレント）
- 会議ごとの GitHub Issue 紐付け（追加/解除、Issue タイトル自動取得）
- GitHub Issue を起点にした会議下書き生成（会議作成ダイアログで会議名・アジェンダ候補を自動入力）
- 下書きアジェンダの取り込み対象をチェック選択して会議作成
- GitHub Issue 入力用テンプレート（`docs/templates/GITHUB_MEETING_INPUT_ISSUE_TEMPLATE.md`）に準拠した取り込み
- リアルタイム音声文字起こし（Web Speech API）
  - 録音ボタンで開始/停止、折りたたみパネルで暫定/確定テキストを表示
  - 確定テキストを現在議題の議事録フィールドへ手動挿入
  - 日本語 / 英語の言語切り替え
  - 非対応ブラウザではボタンをグレーアウト表示
  - AI 音声要約ダイアログ（`VoiceTranscriptSummaryDialog`）
    - 確定済み文字起こしエントリを AI（LangChain / OpenAI・Anthropic）で要約
    - 要約結果を確認・編集して議事録フィールドへ手動挿入
    - AI API 未設定時はルールベース要約へフォールバック
- 個別操作・一括操作

### 会議レポート

- 会議データからレポート草稿を生成
- AIアシスト提案（議事録要約 / 合意形成 / ファシリテーション / 次回アジェンダ / 会議事前準備）を表示し、下書きへ反映（LangChain 経由 OpenAI/Anthropic を使用、AI API 未設定時はルールベース生成へフォールバック）
- AI API 設定（provider/model/api key/temperature/接続テスト）は設定画面から操作できる
- ToDo 管理
- Markdown 出力
- 会議に紐付いた GitHub Issue へのコメント投稿（会議レポート確認ダイアログから手動投稿）
- 差分のみ投稿モード（前回レポートとの差分行のみ Issue コメント投稿）
- Issue チェックリストから ToDo（担当/期限）を自動反映
- 投稿前プレビューと投稿テンプレート選択（要約/詳細）
- 投稿成功した Issue コメント URL の履歴保存

### ログ管理

- レベル別ログ記録（ERROR/WARN/INFO/DEBUG/TRACE）
- LocalStorage 保持
- AI 解析用プロンプト生成

### 分析ダッシュボード

- 日次/週次/月次の集中時間・完了セッション数の可視化
- ポモドーロ達成率、休憩比率、連続達成日数（ストリーク）の表示
- 会議の時間超過率、議題ごとの予定対実績の差分分析
- 時間帯・曜日ごとの生産性ヒートマップ表示
- 目標値（例: 1日120分集中）に対する進捗トラッキング
- フィルタ（期間、タイマー種別、カテゴリ、会議）と比較表示
- CSV/Markdown 形式でのサマリーエクスポート

### MAPE-K 会議効率化（アジェンダ自律最適化）

- **Monitor**: 会議完了時に計画時間と実績時間を自動記録（最大 100 件保持）
- **Analyze**: 3 件以上のデータ蓄積後、アジェンダ項目の超過傾向を分析しパターン学習
- **Plan**: 平均超過率 20% 以上の議題に対して予定時間の修正提案を生成（信頼度スコア付き）
- **Execute**: 提案バッジをアジェンダ編集画面に表示し、ユーザー承認で適用（自動適用なし）
- **Knowledge**: 会議記録・学習パターン・設定を localStorage に永続化
- ローカル計算のみ（外部 API 不要）
- 設定: 提案表示 ON/OFF、学習期間スライダー、データリセット
- 詳細仕様: `.kiro/specs/mape-k-meeting-optimization/`


### 画像 OCR テキスト入力

- ホワイトボードや資料写真からテキストを読み込み、アジェンダ下書きまたは議事録に取り込む
- ファイル選択・ドラッグ&ドロップ・カメラ撮影（モバイル）で画像入力
- ブラウザ内 OCR（Tesseract.js WASM）でサーバー送信なし
- 認識言語: 日本語横書き / 縦書き / 英語
- 認識結果をユーザーが確認・編集してから取り込み（自動反映なし）
- 会議作成ダイアログ: 「画像から読み込む」ボタンでアジェンダ下書きに追記
- 議事録エディタ: カメラアイコンから Quill カーソル位置に挿入
- 詳細仕様: `docs/DESIGN_OCR_IMAGE_INPUT.md`
